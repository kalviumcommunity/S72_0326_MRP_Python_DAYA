🔧 Data Preprocessing

Before analysis, raw data must be cleaned and transformed into a structured format suitable for accurate results.

❌ Before Preprocessing

  • 70,000 encounters with mixed data quality
  • 139,716 missing values across datasets
  • 41,677 invalid dates (59.5% of records)
  • Inconsistent formats in date fields
  • No derived features for analysis
  • Raw categorical data not ready for modeling

✅ After Preprocessing

  • 28,323 clean encounters (40.5% retained)
  • Zero missing values in critical fields
  • Standardized date formats (YYYY-MM-DD)
  • 36 days of aggregated data ready for analysis
  • 5 engineered features (MA_7, MA_14, month, day_of_week, is_weekend)
  • Validated data types and ranges

📋 Preprocessing Steps

1

Data Ingestion

Loaded 3 datasets: CA Hospital (Q1 2025), ICU records, and historical data (2021-2024)

2

Data Cleaning

Removed 41,677 rows with invalid dates, handled missing values, standardized formats

3

Handling Outliers

Identified outliers using box plots, validated extreme values against business rules

4

Feature Engineering

Created moving averages (7-day, 14-day), time features (month, day_of_week, is_weekend)

5

Data Aggregation

Aggregated encounters to daily bed counts, calculated ICU ratio (20.1%)

6

Validation

Verified data quality, checked for consistency, validated date ranges

✅ Result: Clean, structured dataset with 36 days of daily bed occupancy data, ready for time series analysis and forecasting. Data quality improved from 40.5% to 100% usable records.

3️⃣ Identifying Correlations

🔍 Key Correlations Discovered

  • ICU/Emergency visits → Bed occupancy: 20.1% of encounters are ICU/Emergency, directly impacting bed demand
  • Day of week → Occupancy: Weekends show different patterns (Sat: 800.2, Sun: 795.4 avg beds)
  • Monthly seasonality: Peak in September (822.7 beds), lowest in May/June (770.7 beds)
  • Trend direction: Decreasing trend (-0.53 beds/day) suggests improving efficiency or reduced admissions

4️⃣ Forecasting Models

📊 Moving Average Forecast

  • Method: 7-day rolling average
  • Forecast: 777.4 beds/day (constant)
  • Best for: Short-term predictions (1-7 days)
  • Advantage: Simple, smooth, reduces noise
  • Use case: Daily operational planning

📈 Trend-Based Forecast

  • Method: Linear regression extrapolation
  • Forecast: 777.0 → 773.8 beds (decreasing)
  • Best for: Medium-term predictions (7-14 days)
  • Advantage: Captures trend direction
  • Use case: Strategic capacity planning
📊 Forecast Accuracy: R-squared = 0.038 indicates low predictability from linear trend alone. This suggests bed demand is influenced by multiple factors beyond time, requiring more sophisticated models (ARIMA, Prophet) for better accuracy.
