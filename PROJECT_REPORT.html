Project Overview

DAYA analyzes healthcare resource utilization across 60,000 patients and 70,000 encounters to forecast bed demand, identify patterns, and provide actionable recommendations.

Project Goals

  • Forecast bed demand 7-14 days ahead for capacity planning
  • Identify patterns In patient admissions and resource utilization
  • Detect anomalies And potential resource shortages
  • Generate actionable insights For hospital administrators
  • Create alert system For critical resource thresholds

Data Preprocessing

Raw data was cleaned and transformed into a structured format for accurate analysis.

Before Preprocessing

  • 70,000 encounters with mixed quality
  • 139,716 missing values
  • 41,677 invalid dates (59.5%)
  • Inconsistent formats
  • No derived features

After Preprocessing

  • 28,323 clean encounters (40.5%)
  • Zero missing values in critical fields
  • Standardized dates (YYYY-MM-DD)
  • 36 days aggregated data
  • 5 engineered features
Result: Clean, structured dataset with 36 days of daily bed occupancy data, ready for time series analysis and forecasting. Data quality improved from 40.5% to 100% usable records.

Key Insights

Discoveries

  • ICU Impact: 20.1% of encounters are ICU/Emergency, directly impacting bed demand
  • Weekly Patterns: Weekends show different patterns (Sat: 800.2, Sun: 795.4 avg beds)
  • Seasonal Trends: Peak in September (822.7 beds), lowest in May/June (770.7 beds)
  • Trend Direction: Decreasing trend (-0.53 beds/day) suggests improving efficiency

Recommendations

  • Capacity Planning: Use 14-day forecasts for staffing decisions
  • Resource Allocation: Monitor ICU ratio daily for early intervention
  • Alert Response: Establish protocols for critical alerts (>818 beds)
  • Seasonal Preparation: Increase capacity during September peak season
  • Model Enhancement: Implement ARIMA or Prophet for better accuracy
Forecast Accuracy: R-squared = 0.038 indicates low predictability from linear trend alone. This suggests bed demand is influenced by multiple factors beyond time, requiring more sophisticated models for better accuracy.
