Future Roadmap: ML Models

How DAYA enables advanced machine learning

Why This Foundation Matters

The DAYA project creates a robust data pipeline that enables seamless integration of advanced ML models. Our clean data, engineered features, and validation framework are the perfect springboard for sophisticated predictive analytics.

Phase 1: Time Series Models (Q2 2026)

ARIMA

Autoregressive Integrated Moving Average for capturing temporal dependencies and patterns over extended 90-day forecasts.

Impact: 90-day forecasts vs current 7-day

Prophet

Facebook's forecasting tool handling seasonality, holidays, and trend changes with built-in changepoint detection.

Impact: Better seasonal trend capture

✅ Why Ready: Our data already has computed moving averages, seasonality patterns, and 36 days of clean historical data.

Phase 2: Advanced ML Models (Q3 2026)

XGBoost / LightGBM

Gradient boosting models for non-linear pattern detection, achieving 95%+ accuracy on complex relationships.

Impact: 95%+ accuracy (vs 85% baseline)

Random Forest

Ensemble learning for feature importance ranking and identifying key resource demand drivers.

Impact: Feature importance insights

Neural Networks (LSTM)

Long Short-Term Memory networks capturing complex temporal dependencies and long-range patterns in healthcare resource demand.

Impact: Deep learning accuracy on sequences

✅ Why Ready: Our 17+ engineered features provide rich inputs. Moving averages, seasonal indices, and lag features perfect for tree-based models.

Phase 3: Anomaly Detection & Optimization (Q4 2026)

Isolation Forest / LOF

Real-time anomaly detection identifying unusual bed occupancy patterns for proactive alerting.

Impact: Real-time anomaly alerts

Clustering (K-Means)

Patient segmentation for intelligent resource allocation based on care intensity and demand patterns.

Impact: Segment-based optimization

Optimization Algorithms

Genetic algorithms and constraint programming for optimal staff scheduling and bed assignment automation.

Impact: 30% efficiency gain

✅ Why Ready: Our alert zones (Green/Yellow/Red) provide labeled training data. Threshold values enable supervised model training.
